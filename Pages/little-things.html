Project Overview

Context

Little Things is a 2D puzzle-platformer built around tight movement and color-based interactions. The project prioritizes mechanical clarity, fair challenge, and a smooth learning curve through structured level progression.

Goals

  • Deliver responsive movement and consistent player feedback.
  • Build scalable mechanics with clean, reusable systems.
  • Create levels that teach through readability and pacing.

Gameplay & Design Focus

Core Mechanics

  • Precision platforming (jump timing, spacing, momentum).
  • Color-based interactions affecting traversal and puzzles.
  • Progression built around mastery, not randomness.

Design Pillars

  • Clarity & fairness: failures communicate learning.
  • Flow: challenge curve tuned to reduce friction.
  • Readability: strong visual cues for player decisions.

Technical Implementation

Architecture

  • Decoupled systems with clear responsibilities.
  • State-driven player controller for consistent behavior.
  • Event-based interactions for scalable mechanics.

Key Systems

  • Player controller + movement tuning tools.
  • Color interaction framework.
  • Level/trigger system for pacing and progression.
Highlight: Systems were designed to be reusable across levels and easy to iterate on during production without rewriting core gameplay code.
