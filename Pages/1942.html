Project Overview

Context

This project is a structured remake of the classic 1942 arcade game. The focus is not only recreating the loop, but building maintainable gameplay architecture as if it were a small engine-driven project.

Goals

  • Apply SOLID principles and clean code practices.
  • Separate gameplay responsibilities (input, update, rendering).
  • Implement robust scene/state management.

Gameplay & Design Focus

Core Loop

  • Arcade shooter loop with scoring and progression.
  • Player movement + shooting + enemy patterns.
  • Clear state transitions: menu → gameplay → results.

Design Pillars

  • Predictable rules and readable collisions.
  • Consistent update order and deterministic behavior.
  • Clear separation between game states.
