Problem & Goals

Problem

Coaches and athletes need clear ways to evaluate motion quality. This prototype explores how to capture movements with Rokoko, animate a 3D model, and compare user motion against professional references for training feedback.

Goals

  • Integrate Rokoko motion data into Unreal in a stable pipeline.
  • Visualize and compare user motion against reference animations.
  • Support fast iteration on metrics, playback, and review UX.

System Overview

Key Features

  • MoCap capture → retargeting → animation playback.
  • Side-by-side comparison (user vs pro).
  • Review controls (scrub, slow motion, pose inspection).

Pipeline / Flow

  • Capture session → import/stream data
  • Retarget to skeleton → save animation clip
  • Compare playback + metrics overlay
